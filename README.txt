Birthday Calculator Demo

    To run this application you will need:
        Java JDK installed and the JAVA_HOME environment variable pointed to the java folder location on your computer
        Maven installed and the MAVEN_HOME environment variable pointed to the maven folder location on your computer


    Clone this project onto your local machine
    In your terminal run
        mvn spring-boot:run
    or
        mvn spring-boot:build-image
        java -jar <target/jar_file>
